自家製TRPGログコンバーターツールです。
更新は気まぐれなので期待しないで下さい。

現在ココフォリアとUdonarium(及びUdonarium with Fly)のログ整形が可能です。
何かあっても自己責任で

使用方法
ココフォリアのログ整形
	前提として、秘匿チャットはタブ名に「秘匿」か「Secret」という文字が含まれている場合は色付けされます。
	1.「ファイルタイプ選択」よりココフォリアを選択
	2.「ファイル選択」ボタンを押下し、開かれたファイル選択画面でログ原本のHTMLを選択する。
		なおこの際、ドラッグアンドドロップでも指定が可能です。
	3.「解析」ボタンを押下する。
	4.画面下部に対応するキャラクター名一覧が表示されます。
	5.アイコンを付けたいキャラクターが存在する場合は該当キャラクターと同行の「ファイル選択」ボタンを押下し、開かれたファイル選択画面で「png、jpg、jpeg」形式のファイルを選択して下さい。
		(画面上にカラーコード選択機能がありますが、現在未実装となっています。)
		なおこの際、ドラッグアンドドロップでも画像追加が可能です。
		2枚目以降の画像を追加した場合、後述の詳細解析機能のためのリザーブ画像となるのでご注意ください。
			どうしても変更したい場合は対象キャラクター名で右クリック→デフォルト画像差し替えを選択していただくと、画像選択ウィンドウが表示されるので、デフォルト画像を変更することができます。
	6.「出力」ボタンを押下すると、HTML形式に整形され、保存場所を求められます。

Udonarium(Udonarium with Fly)のログ整形
	前提として、「メイン」という名前のタブとそれ以外を対象としています。
	また、秘匿チャットは非対応です。対応予定もありません。
	1.「ファイルタイプ選択」よりユドナリウムを選択
	2.「ファイル選択」ボタンを押下し、開かれたファイル選択画面でログ原本のHTMLを選択する。
		なおこの際、ドラッグアンドドロップでも指定が可能です。
	3.「解析」ボタンを押下すると、HTML形式に整形され、保存場所を求められます。
		この際、元データがUdonarium with Flyで、スタンド解析にチェックを入れている場合、スタンドの画像が割り当てられて解析されます。(※注意:ファイルサイズが増大します！)
		なお、詳細解析チェックを入れている場合は自動出力はされず、ココフォリア同様「出力」ボタンの押下が必要となります。

詳細解析
	チャットメッセージ毎に画像を差し替え可能です。
		対象メッセージ項目をダブルクリックすると画像選択ウィンドウが表示されます。
		そのウィンドウにドラッグアンドドロップで画像を追加すると変更可能となります。
			なお、この際選択後すべての同名キャラクターの画像を差し替えることも可能です。
	上部ボタンや、右クリックでイベント画像などの枠追加も可能です。
